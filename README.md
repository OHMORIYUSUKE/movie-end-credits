# Movie End Credits (Remotion)

映画「【劇場版】アニメから得た学びを発表会2026」のエンドロール動画を自動生成するプロジェクトです。
左側に投稿された写真のスライドショー、右側にスタッフロールがスクロール表示されます。

---

## 🚀 システム構成

本プロジェクトは2つの主要コンポーネントで構成されています。

1.  **エンドロール生成エンジン (Remotion)**: `src/`
    *   Reactベースで動画をレンダリングします。
    *   アップロードされた写真から自動的に選出し、時系列順に表示します。
2.  **写真投稿・管理システム (Deno Fresh)**: `web/`
    *   参加者がスマホから写真を投稿できるウェブサイトです。
    *   管理者が写真の選別や優先表示設定を行える管理パネルを備えています。

---

## 📸 写真アップロード・管理 (Deno Fresh)

### 起動方法
```bash
cd web
deno task start
```
起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

### 写真の取り扱い
- アップロードされた写真は、[【劇場版】アニメから得た学びを発表会2026](https://engineers-anime.connpass.com/event/375981/) のイベント内や広報で利用される可能性があります。
- 収集された写真は、イベントの [プライバシーポリシー](https://fortee.jp/engineers-anime-2026/page/privacy-policy) に基づいて適切に取り扱われます。

### 管理機能 (`/login` からアクセス)
- **承認制**: ユーザーが投稿した写真は初期状態で「非表示」です。管理者が「表示する」に切り替えた写真のみが動画の候補になります。
- **必ず採用 (🌟)**: 特定の写真を確実に動画に含めたい場合に設定します。
- **自動集計**: 動画の長さに合わせた「収容枚数」と現在の「採用状況」がリアルタイムに表示されます。
- **EXIF対応**: 画像に含まれるEXIFデータから撮影日時を自動抽出し、時系列順に並べ替えます。

### 環境設定
`web/.env` ファイルで設定を変更できます。
- `ADMIN_PASSWORD`: 管理者パスワード。

## 🎬 エンドロール生成 (Remotion)

### 1. セットアップ
```bash
npm install
```

### 2. データの同期
Deno Fresh 側で管理された写真データは、レンダリング時に API 経由で取得されます。
動画の尺（音楽の長さ）やクレジット（名前）などのローカルデータを最新の状態に更新するには、以下のコマンドを実行します。

```bash
# クレジットと動画の尺をまとめて更新
# ※音楽（public/music/end.mp3）の長さに合わせて動画の尺を自動調整し、
#   最新の登壇者・スタッフ・スポンサー情報を取得します。
npm run update
```

### 3. 設定ファイルについて
`src/generated/` 配下の設定ファイルは、`npm run update` によって自動生成されます。
初回は `npm install` 時に `.example.json` から自動的に初期化されます。

- `src/generated/video-config.json`: 動画の尺やFPSの設定。
- `src/generated/credits.json`: スタッフロールに表示される名前データ。

### 4. 採用ロジック
動画に表示される写真は以下のルールで選出されます。
1.  管理者が「表示」に設定した写真が対象。
2.  「必ず採用」に設定された写真をすべて抽出。
3.  残りの枠を、その他の対象写真からランダムに補充（枚数は動画の長さに依存）。
4.  選考された写真は、**枚数に応じて表示間隔が自動調整**され、尺に収まるように表示されます。
5.  選出された写真を**撮影日時の早い順**に並べて表示。

### 5. プレビューとレンダリング
```bash
# ブラウザでプレビューを確認
npm run dev

# MP4形式で動画を書き出し
npx remotion render EndCredits out.mp4
```

---

## ⚠️ 注意事項
- **OS依存**: `npm run update` コマンド（動画尺の取得）は macOS 標準の `afinfo` を使用します。
- **Git管理**: `src/generated/*.json` は自動生成されるため Git 管理外です。デフォルト値は `*.example.json` に保存されています。
- **写真の管理**: 以前のローカルファイル（`public/photos/`）による管理は廃止されました。現在はウェブシステム経由で Deno KV に保存されます。
