# Movie End Credits (Remotion)

映画「【劇場版】アニメから得た学びを発表会2026」のエンドロール動画を自動生成するプロジェクトです。
左側に投稿された写真のスライドショー、右側にスタッフロールがスクロール表示されます。

---

## 🚀 システム構成

本プロジェクトは2つの主要コンポーネントで構成されています。

1.  **エンドロール生成エンジン (Remotion)**: `src/`
    *   Reactベースで動画をレンダリングします。
    *   アップロードされた写真から自動的に選出し、時系列順に表示します。
2.  **写真投稿・管理システム (Deno Fresh)**: `web/`
    *   参加者がスマホから写真を投稿できるウェブサイトです。
    *   管理者が写真の選別や優先表示設定を行える管理パネルを備えています。

---

## 📸 写真アップロード・管理 (Deno Fresh)

### 起動方法
```bash
cd web
deno task start
```
起動後、ブラウザで `http://localhost:8000` にアクセスしてください。

### 写真の取り扱い
- アップロードされた写真は、[【劇場版】アニメから得た学びを発表会2026](https://engineers-anime.connpass.com/event/375981/) のイベント内や広報で利用される可能性があります。
- 収集された写真は、イベントの [プライバシーポリシー](https://fortee.jp/engineers-anime-2026/page/privacy-policy) に基づいて適切に取り扱われます。

### 管理機能 (`/login` からアクセス)
- **承認制**: ユーザーが投稿した写真は初期状態で「非表示」です。管理者が「表示する」に切り替えた写真のみが動画の候補になります。
- **必ず採用 (🌟)**: 特定の写真を確実に動画に含めたい場合に設定します。
- **自動集計**: 動画の長さに合わせた「収容枚数」と現在の「採用状況」がリアルタイムに表示されます。
- **EXIF対応**: 画像に含まれるEXIFデータから撮影日時を自動抽出し、時系列順に並べ替えます。

### 環境設定
`web/.env` ファイルで設定を変更できます。
- `ADMIN_PASSWORD`: 管理者パスワード。

## 🎬 エンドロール生成 (Remotion)

### 1. セットアップ
```bash
yarn install
```

### 2. アセットの配置
動画で使用するロゴは以下の場所に配置してください。

- **ロゴ**: `public/logo/logo.svg`
  - スタッフロールの最後に表示されます。

### 3. 背景音楽と動画の尺
動画の長さは、**管理システムで「使用する」に設定された音楽の長さ**に合わせて自動的に調整されます。

- **ウェブ管理画面から設定**:
  - 管理パネル（`/login`）の「音楽管理」からMP3をアップロードし、「使用する」ボタンでアクティブな曲を選択します。
  - アップロード時に自動的に尺（秒数）が計測され、動画のレンダリングに反映されます。

### 4. クレジットの更新
名簿リスト（エンドロールの名前）を更新するには、以下の手順で行います。

1. **JSONへの変換**:
   `public/credit/` 内のCSVから `credits.json` を生成します。
   ```bash
   npm run convert-credits
   ```
2. **アップロード**:
   管理パネル（`/login`）の「クレジット管理」から、生成された `credits.json` をアップロードします。

---

### 5. プレビューとレンダリング
```bash
# ブラウザでプレビューを確認（Denoサーバーが起動している必要があります）
npm run dev

# MP4形式で動画を書き出し
npx remotion render EndCredits out.mp4
```

- **ロックファイル**: 本プロジェクトは `yarn` を使用しています。
