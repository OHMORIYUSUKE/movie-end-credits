# Movie End Credits (Remotion)

映画「【劇場版】アニメから得た学びを発表会2026」のエンドロール動画を自動生成するプロジェクトです。
左側に投稿された写真のスライドショー、右側にスタッフロールがスクロール表示されます。

---

## 🚀 システム構成

本プロジェクトは3つの主要コンポーネントで構成されています。

1.  **エンドロール生成エンジン (Remotion)**: ルートディレクトリ
    *   Reactベースで動画をレンダリングします。
    *   アップロードされた写真から自動的に選出し、時系列順に表示します。
2.  **API サーバー (Deno Fresh)**: `api/`
    *   Deno KV を使用したデータ永続化と、画像・音声の処理を担当します。
    *   Next.js からのリクエストを受け付けるバックエンドです。
3.  **管理・投稿画面 (Next.js)**: `frontend/`
    *   App Router + Tailwind CSS で構築されたユーザーインターフェースです。
    *   参加者の写真投稿画面と、管理者の選別パネルを提供します。

---

## 📸 起動方法

### 1. バックエンド (Deno API)
```bash
cd api
deno task start
```
デフォルトで `http://localhost:8000` で起動します。

### 2. フロントエンド (Next.js)
```bash
cd frontend
yarn dev
```
起動後、ブラウザで `http://localhost:3000` にアクセスしてください。
※ `/api/*` へのリクエストは自動的に Deno API にプロキシされます。

---

## 🛠️ 管理機能と設定

### 管理パネルへのアクセス
- `http://localhost:3000/login` からアクセスできます。
- **管理者パスワード**: `api/.env` の `ADMIN_PASSWORD` で設定します（デフォルト: `admin123`）。

### 主な機能
- **写真の選別**: ユーザーが投稿した写真は初期状態で「非表示」です。管理者が「採用」に切り替えた写真のみが動画の候補になります。
- **必ず採用 (🌟)**: 特定の写真を確実に動画に含めたい場合に設定します。
- **音楽・ロゴ・クレジット管理**: ウェブ画面から MP3、ロゴ画像、`credits.json` をアップロードして動画の内容をリアルタイムに更新できます。

## 🎬 エンドロール生成 (Remotion)

### セットアップ
```bash
yarn install
```

### プレビューとレンダリング
```bash
# ブラウザでプレビューを確認（Deno APIが起動している必要があります）
npm run dev

# MP4形式で動画を書き出し
npx remotion render EndCredits out.mp4
```

## ⚠️ 注意事項
- **写真の管理**: 全て Deno KV に保存されます。
- **CORS**: `api/` 側で Next.js からのアクセスを許可するように設定されています。
